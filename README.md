# 🚀 Keen Keeper

Keen Keeper is a modern web application built to manage and visualize friend-related data in an interactive and insightful way. It focuses on delivering a clean user experience with dynamic data fetching, timeline tracking, and visual analytics.

---

## 📌 Overview

Keen Keeper allows users to:

* Fetch and explore friend data
* View detailed information about individuals
* Track activities through a timeline system
* Visualize data using charts

The project emphasizes performance, simplicity, and responsive design.

---

## 🛠️ Tech Stack

This project is built using:

* **React.js** – Component-based UI development
* **Next.js** – Full-stack React framework for optimized performance
* **JavaScript (ES6+)** – Core programming language
* **Tailwind CSS** – Utility-first CSS framework for styling
* **React Router** – Client-side routing
* **React Icons** – Icon library for UI enhancement
* **Recharts / Chart Utilities** – Data visualization

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate to project directory
cd keen-keeper

# Install dependencies
npm install

# Run the development server
npm run dev
```

## 🚀 Usage

* Open your browser and go to: `http://localhost:3000`
* Explore friend data
